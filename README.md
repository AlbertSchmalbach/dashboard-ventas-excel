# 📊 Dashboard de Ventas en Excel

## 🚀 Descripción
Proyecto de análisis de datos desarrollado en Excel para visualizar y automatizar información de ventas.

## 🧠 Funcionalidades
- Análisis de ventas por producto, ciudad y vendedor
- Dashboard interactivo con segmentadores
- Automatización con macros VBA
- Indicadores clave (KPI)

## 🛠️ Herramientas utilizadas
- Excel avanzado
- Tablas dinámicas
- VBA (macros)

## 📈 Objetivo
Demostrar habilidades en análisis de datos y automatización de procesos.
